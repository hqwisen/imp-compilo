# imp-compilo

Tested with Java 8 (JDK)

# How To

```bash
gradle run # Run Main class
gradle jar # Create dist/imp-compilo.jar with Main as Main Class
```

## Dependencies

* make
* jflex

```bash
apt install jflex # Ubuntu
pacman -S jflex # Arch Linux
```
